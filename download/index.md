OS X
----

We support [homebrew][homebrew], a package manager for OSX. To install
dReal, run the following scripts on a terminal:

{% highlight bash %}
brew tap dreal/dreal
brew install dreal
{% endhighlight %}


To upgrade dReal, run:

{% highlight bash %}
brew upgrade dreal
{% endhighlight %}

[homebrew]: http://brew.sh/
